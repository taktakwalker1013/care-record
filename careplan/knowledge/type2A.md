# ケアプランテンプレート：タイプ②-A
## 自宅（独居）＋介護保険＋重度訪問介護（障害福祉サービス）併用型

> **作成日**：令和8年5月14日
> **適用対象**：自宅で独居生活を継続しつつ、介護保険サービスを基本としながら、不足分を障害福祉サービスの重度訪問介護で補完して24時間ケアを実現する利用者

---

## 1. 適用対象

以下の条件を**すべて**満たす利用者に適用する。

| 条件 | 内容 |
|---|---|
| 生活拠点 | 自宅（独居） |
| 介護保険 | 要介護認定あり（要介護1以上、典型は要介護4〜5） |
| 障害福祉 | 障害支援区分4以上、かつ重度訪問介護の対象要件を満たす |
| 年齢要件 | **65歳到達前から重度訪問介護を利用していた者**が継続利用するケースが原則 |
| 必要なケア量 | 介護保険の区分支給限度額のみでは24時間ケアが組めない |
| 居住環境 | 自宅での生活継続が可能な住環境（住宅改修済み or 改修可能） |

### 重度訪問介護の対象要件（参考）

- 障害支援区分4以上
- 二肢以上に麻痺等があり、障害支援区分認定調査項目のうち「歩行」「移乗」「排尿」「排便」のいずれも「支援が不要」以外と認定されている
- または、行動関連項目等の合計点数が10点以上

### 自宅独居の前提条件

- 緊急時連絡先（家族・親族・後見人等）が確保されていること
- 住環境が利用者の身体状況に適合していること、または改修により適合可能であること
- 重度訪問介護を提供できる事業所が地域に複数存在すること（単一事業所では24時間対応が困難）

---

## 2. 制度的論点

### 介護保険サービスの優先原則（障害者総合支援法第7条）

介護保険と障害福祉で**同等のサービス**がある場合、原則として介護保険が優先される。ただし、介護保険サービスでは支援が**不足する場合**、その不足分について障害福祉サービスを併給できる。

### 重度訪問介護の65歳問題

65歳到達後に新規で重度訪問介護を取得することは原則困難。**65歳前から継続利用している者**が引き続き利用するケースが典型。本テンプレートはこの「継続利用組」を主な対象とする。

### 自宅独居での重訪利用の留意点

- **施設のような24時間常駐職員がいない**ため、訪問サービスの空白時間を作らない設計が必須
- **緊急時の初動対応**は重訪ヘルパーが第一線、医療職や家族はバックアップという役割分担
- **複数事業所による連携体制**が前提（単一事業所での24時間カバーは困難）
- **生活援助の比重が高い**（調理、買物、洗濯、掃除等を介護保険または重訪で実施）

### 区分支給限度額との関係

要介護5の区分支給限度基準額は36,217単位/月（令和8年現在）。介護保険サービスは限度額内に収め、不足分を重度訪問介護で補完するのが基本構造。重度訪問介護は障害福祉サービスのため、介護保険の区分支給限度額には含まれない。

## 3. サービス組み合わせ（基本仕様）

### 基本訪問頻度

**24時間切れ目のない訪問体制**を構築する。介護保険サービスを「固定枠」として配置し、その他の時間帯を重度訪問介護で連続的にカバーする。

### 24時間配分の例

| 時間帯 | 担い手 | 主な内容 |
|---|---|---|
| 0:00〜5:00 | 重度訪問介護 | 夜間見守り、体位変換、排泄介助 |
| 5:00〜8:00 | 重度訪問介護 | 起床準備、身支度補助 |
| 8:00〜9:30 | **訪問介護**（身体2生活1） | 朝の身体介護、朝食、服薬支援、片付け |
| 9:30〜12:00 | 重度訪問介護 | 見守り、コミュニケーション支援 |
| 12:00〜13:00 | 重度訪問介護 | 昼食介助 |
| 13:00〜15:00 | 重度訪問介護 | 見守り、外出支援、家事支援 |
| 15:00〜18:00 | **訪問介護**（身体2生活1）+ 重度訪問介護 | 午後の身体介護、入浴介助、夕食準備 |
| 18:00〜21:00 | 重度訪問介護 | 夕食介助、就寝準備 |
| 21:00〜翌0:00 | 重度訪問介護 | 就寝後の見守り、夜間排泄 |

※時刻・配分は目安。利用者の生活リズム、各事業所の対応可能時間に応じて柔軟に調整可。

### 自宅独居型に多い「身体介護＋生活援助」の長時間訪問

自宅独居型では、施設のように食事提供がないため、訪問介護で**身体介護と生活援助を組み合わせた長時間訪問**を組むことが多い。代表的なパターン：

| 報酬区分の通称 | 構成 | 所要時間目安 | 主な内容 |
|---|---|---|---|
| **身体1生活1** | 身体30〜60分＋生活20〜45分 | 約60〜90分 | 朝の身体介護＋簡単な家事 |
| **身体2生活1** | 身体60〜90分＋生活20〜45分 | 約90分〜2時間 | 入浴介助＋食事準備等 |
| **身体3生活1** | 身体90〜120分＋生活20〜45分 | 約2〜2.5時間 | 重度の身体介護＋家事 |

> **記載例の見方**：「身体1生活1」は「身体介護1単位＋生活援助1単位の組み合わせ」を指す業界略称。最初の数字が身体介護の時間区分、後ろの数字が生活援助の時間区分を表す。

身体介護と生活援助を**同一訪問内**で連続実施することで、訪問の都度の出入りを減らし、効率的にケアが提供できる。重訪が前後の時間帯を埋めるため、引継ぎが円滑に行える設計となる。

### 介護保険サービスの枠割り

| サービス | 頻度の目安 | 役割 |
|---|---|---|
| 訪問介護（朝枠） | 1日1回 | 朝の身体介護、服薬、朝食の中核 |
| 訪問介護（午後枠） | 1日1回 | 入浴介助、午後の身体介護 |
| 訪問看護 | 週1〜3回 | 医療的ケア、状態観察、褥瘡処置 |
| 訪問入浴介護 | 週1〜2回 | 入浴介助（必要に応じ） |
| 訪問リハビリ | 週1〜2回 | 状態維持・嚥下評価が必要な場合に検討 |
| 居宅療養管理指導 | 月1〜2回 | 訪問診療、薬剤師訪問、歯科訪問 |
| 福祉用具貸与 | 随時 | 特殊寝台、エアマットレス、車いす等 |
| 居宅介護支援 | 月1回 | ケアマネジメント |

### 重度訪問介護のシフト構成

24時間を複数事業所でシフト交代制に分担するのが現実的。単一事業所での24時間カバーは事業所側の負担が大きく、継続性に課題が生じやすい。

| 観点 | 推奨される運用 |
|---|---|
| 事業所数 | 3〜5事業所が標準、利用時間が長いほど事業所数も増える |
| 時間帯分担の固定化 | 利用者・ヘルパー双方の慣れのため、できるだけ固定 |
| 引継ぎ | 事業所間で記録・申し送りのルールを統一 |
| バックアップ体制 | 1事業所が欠勤・休業の場合の代替体制を確保 |

### 役割分担の原則

| 担い手 | 担う内容 |
|---|---|
| 訪問介護（介護保険） | 身体介護の中核（入浴介助、服薬支援等の専門性が必要なケア）＋生活援助（朝・夕の食事準備等） |
| 重度訪問介護（障害福祉） | 長時間の見守り、定時の体位変換、排泄介助、コミュニケーション支援、家事援助・生活支援 |
| 訪問看護（介護保険） | 医療的ケア、状態観察、褥瘡処置等の医療職判断が必要なケア |
| 訪問入浴介護（介護保険） | 入浴専用機材による全身入浴（自宅浴室での入浴が困難な場合） |
| 訪問リハビリ（介護保険） | 状態維持・嚥下評価・関節拘縮予防等のリハビリテーション |
| 居宅療養管理指導（介護保険） | 訪問診療、薬剤師訪問、歯科訪問による医療・服薬・口腔の管理 |
| 福祉用具貸与（介護保険） | 特殊寝台、エアマットレス、車いす等の貸与による生活環境整備 |

## 4. ニーズ設定の典型パターン

このタイプで頻出するニーズ：

1. **24時間の安全な見守りと支援を受けたい**（最重要）
2. **医療的ケア（疾患・障害の特性に応じた処置）を継続したい**
3. **長年利用してきた重度訪問介護のヘルパーとの関係を維持したい**
4. **障害特性（コミュニケーション方法、こだわり等）に配慮した支援を受けたい**
5. **自宅での生活を継続したい**（独居維持の意思尊重）
6. **緊急時に速やかな対応を受けたい**（救急要請・家族連絡の体制）
7. **食事・洗濯・掃除等の生活援助を継続的に受けたい**（独居特有）
8. **社会的孤立を防ぎ、人との関わりを保ちたい**（独居特有）

### 医療依存度別のバリエーション

このタイプ②-Aは、医療依存度の高低によってニーズの重点が大きく変わる。利用者の状態に応じて、以下のバリエーションを参考にする。

#### バリエーションの選び方

- 利用者の主たる障害・疾患特性に応じて、いずれかのバリエーションを参照する。
- 単一のバリエーションに収まらない場合は、**複数のバリエーションを組み合わせて適用**する（例：医療依存高型＋認知症併存型 など）。
- 状態変化により該当バリエーションが変わることもあるため、定期的に見直す。

#### バリエーション①：医療依存高型

該当例：気管切開、人工呼吸器、経管栄養、頻回の吸引、褥瘡処置等が日常的に必要。

| 重点的に検討すべき項目 | 対応の方向性 |
|---|---|
| 医療的ケアの継続 | 訪問看護の頻度を週2〜3回以上に。緊急時24時間対応契約 |
| 医療職との連携 | 在宅医・訪問看護師・薬剤師の多職種カンファレンス定期化 |
| 重訪ヘルパーの医療的ケア研修 | 喀痰吸引等研修の修了者を配置 |
| 緊急入院時の対応体制 | 受入医療機関、搬送経路、家族連絡を事前確認 |

#### バリエーション②：医療依存低型

該当例：身体機能障害が中心、医療処置は服薬管理程度。

| 重点的に検討すべき項目 | 対応の方向性 |
|---|---|
| 生活援助・コミュニケーション支援 | 食事・洗濯・掃除・買物の比重が高い |
| 社会参加・夕出支援 | 同行援護・移動支援との連携、外出機会の確保 |
| 認知機能の維持 | 会話・趣味活動の継続、見守りの中での自立支援 |
| 福祉用具・住環境整備 | 状態変化に応じた機動的な見直し |

#### バリエーション③：認知症併存型

該当例：身体障害に加え、**加齢に伴い認知症（アルツハイマー型・血管性等）が併発した**ケース。

| 重点的に検討すべき項目 | 対応の方向性 |
|---|---|
| 周辺症状（BPSD）への対応 | 重訪ヘルパーへの認知症ケア研修、対応方法の統一 |
| 服薬管理の徹底 | 訪問看護師・薬剤師との連携、服薬カレンダー等の活用 |
| 安全確保（火気・施錠等） | センサー類の活用、訪問の空白時間を作らない |
| 意思決定支援 | 成年後見制度の検討、家族・後見人との連携 |

## 5. 注意点・落とし穴

### 区分支給限度額の管理

訪問介護を朝・夕の2回/日（月60回前後）にしても、要介護5の限度額内で十分収まる範囲。**訪問看護や福祉用具を含めた合計単位**を必ず試算する。重度訪問介護で大部分の時間を埋めるため、介護保険サービスは限度額の50〜70%程度に収まる設計が標準。

### 役割分担の明文化

「同じ時間帯に重度訪問介護と訪問介護が重複していないか」「両者の役割が明確に区別されているか」を、第2表の援助内容欄で明示する。**保険者からの問い合わせ時の説明根拠**として重要。

### ケアマネと相談支援専門員の連携

ケアマネ（介護保険）と相談支援専門員（障害福祉）の**両方が関与**する。サービス担当者会議に相談支援専門員も呼び、サービス等利用計画とケアプランの整合性を取る。重訪事業所が複数の場合、すべての事業所が会議に参加することが理想。

#### 役割分担の目安

| 担当 | 主な作成書類 | 主な対応範囲 |
|---|---|---|
| **ケアマネ** | 居宅サービス計画書（第1〜7表）、モニタリング記録 | 介護保険サービス全般、医療連携、住環境 |
| **相談支援専門員** | サービス等利用計画案、モニタリング報告書（障害福祉側） | 重度訪問介護等の障害福祉サービス全般、障害特性への配慮 |
| **両者共同** | サービス担当者会議の開催・調整 | サービス全体の整合性確保、緊急時対応の方針共有 |

#### 連絡頻度の目安

- 通常時：月1回程度の情報共有
- 状態変化時・サービス調整時：その都度
- 担当者会議：年2〜4回程度（プラン更新時、状態変化時等）

### 重度訪問介護の事業所選定と連携体制

- **長時間対応可能な事業所**を時間帯別に確保
- 各事業所間の**記録・申し送りルール**を統一
- **1事業所欠勤時のバックアップ体制**を事前に確認
- 従来から関わっている事業所を軸としつつ、24時間体制に必要な事業所は新規にも開拓する
- 事業所ごとに対応可能な曜日・時間帯が異なるため、**全体のシフト表を1枚にまとめて空白時間を可視化**する
- ヘルパー不足により希望時間帯を埋められないことがあるため、**支給量に余裕を持たせた申請**と早めの調整を行う

### 緊急時対応体制（自宅独居特有）

- 明らかな急変時はヘルパー判断で救急要請可、判断に迷う場合は訪問看護師に第一報、という二段階の基準を設定する
- 救急要請の判断基準を**重訪ヘルパー・訪問看護師と共有**
- キーパーソンの居住地（近隣/遠方）に応じて、駆けつけ依頼の可否・到着所要時間を整理する
- 家族（キーパーソン）への連絡フローを明確化
- 在宅医・訪問看護ステーションの**24時間連絡先**を確保
- 緊急通報装置・センサー等の活用を検討
- 入院に至った場合のサービス一時停止手続き・退院支援の流れも事前に整理しておく

### キーパーソンとの連携体制（自宅独居特有）

キーパーソンの居住状況により留意点が異なる。以下の3パターンに整理して対応する。

| パターン | 留意点 |
|---|---|
| **近隣に居住** | 緊急時の駆けつけが可能で心強い反面、家族の負担過多や、家族とヘルパーの役割境界の曖昧化に注意。家族の高齢化により対応力が低下する可能性も見込む |
| **遠方に居住** | 地域包括支援センターや民生委員等の地域資源を活用。緊急時の連絡フローと、家族到着までの初動を重訪ヘルパー・訪問看護師で完結できる体制を整える |
| **家族・親族が不在** | 後見人（成年後見・任意後見）の有無を確認。意思決定支援の体制を整え、地域包括支援センターを中心とした支援ネットワークを構築する |

共通の留意点：
- 後見人（成年後見・任意後見）の有無を確認
- 入院時の**身元保証人の確保**について事前に整理（家族・後見人・成年後見制度の活用等）

### 住環境の継続的な見直し

- 状態変化に応じた住宅改修の必要性を定期的に評価
- 福祉用具の追加・変更を機動的に対応
- 廊下幅・段差・浴室・トイレ等のバリアフリー状況を点検
- 賃貸住宅の場合は改修に大家の承諾が必要なため、可動式の福祉用具での代替も検討する
- 介護保険・障害福祉の双方に住宅改修制度があり、対象や上限が異なるため、相談支援専門員と連携して有利な制度を選択する

## 6. 第1表「総合的な援助の方針」記載例

> 介護保険サービスと障害福祉サービス（重度訪問介護）を組み合わせ、24時間切れ目のない支援体制を構築し、利用者が自宅での独居生活を継続できるよう支援する。専門性が必要な身体介護・医療的ケアは介護保険サービスで対応し、その間の見守り・状態確認・コミュニケーション支援・家事援助・生活支援は重度訪問介護で拆う。複数の重訪事業所が連携し、いずれの時間帯でも切れ目のない支援を提供する。ケアマネと相談支援専門員が連携し、サービス全体の整合性を保ちつつ、緊急時には在宅医・訪問看護師・家族（キーパーソン）との連絡体制を確保する。

## 7. 関連する障害福祉サービス・社会資源（必要に応じ追加検討）

### 障害福祉サービス

- 行動援護
- 同行援護
- 短期入所（ショートステイ）
